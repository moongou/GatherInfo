# 实施路线图

## 0-2 周: 可演示原型

- 接入 5-10 个公开官方源。
- 建立来源登记、采集哈希、字段抽取与前端驾驶舱。
- 完成 TBT/SPS 问答与日报生成闭环。

## 1-3 个月: 一期基座

- 扩展 50 个核心明网站点。
- 引入队列、对象存储、全文索引与图数据库。
- 建立 HS Code、机构、企业、口岸主数据。
- 上线分析师复核与签发流程。

## 4-7 个月: 能力跃升

- 扩展至 1000+ 信源。
- 接入授权 AIS、企业征信、商业贸易数据库。
- 上线走私态势感知与战术线索推送。
- 引入多语种 ASR 与视频文本抽取。

## 8-12 个月: 预测推演

- 上线时序预测、图嵌入和多智能体沙盘。
- 形成季度战略报告与行业专题自动生成链路。
- 建立模型评测、漂移检测和人在回路微调闭环。
